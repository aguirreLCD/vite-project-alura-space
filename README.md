# Get started:

**Alura Space** is a project bootstrapped with create vite@latest:

```
npm create vite@latest
```

Select:

```
- name: vite-project-alura-space
- framework: React
- variant: Javascript
```

Installation:

`npm install`

To Start Server:

`npm run dev`

To Visit App:

`localhost:5173/`

Open `http://localhost:5173` with your browser to see the application running.
